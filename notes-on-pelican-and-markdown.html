Notes on Pelican and markdown

Sat 14 June 2014
by elfnor
in make

tagged make, notes only

I'm now using Pelican to generate this blog from content stored in plain tect files. It's installation is well explained on the Pelican website and many blogs (the one I used). I used the method where you set up a virtualenv and install pelican into that.

The only problem I had was that after I'd cloned the standard themes from github:

git clone --recursive https://github.com/getpelican/pelican-themes ~/pelican-themes

I couldn't get the path to the themes ~/pelican-themes to work in the pelicanconf.py file. The hack solution was to copy all the theme folders and files from ~/pelican-themes/ to ~/virtualenvs/lookthinkmake/lib/python2.7/site-packages/pelican/themes/ and use the following line in pelicanconf.py. 

THEME = 'pelican-bootstrap3'

where the folder of the desired theme is specified.

To restart a devlopment session:

cd ~/lookthinkmake
source ~/virtualenvs/lookthinkmake/bin/activate
make devserver

and point firefox to localhost:8000.

I also swapped this version of server.py for the one in ~/virtualenvs/lookthinkmake/lib/python2.7/site-packages/pelican/ to fix some Broken Pipe Errors.

The use of references for links looks like good practice.

A good cheat sheet.

Tables kind of work without a plugin, layout may be theme dependent.

Pipes can be used to align columns.

This site can be used to generate markdown tables from a copy and paste from a spreadsheet etc.

The outer pipes (|) are optional, and you don't need to make the raw Markdown line up prettily. You can also use inline Markdown.

The look of the table is defined in the main.css file. 

table {
    font-size:0.9em;
    color:#333333;
    border-width: 1px;
    border-color: #666666;
    border-collapse: collapse;
    width: 100%;
}
table th {
    border-width: 1px;
    padding: 8px;
    border-style: solid;
    border-color: #666666;
    background-color: #dedede;
}
table td {
    border-width: 1px;
    padding: 8px;
    border-style: solid;
    border-color: #666666;
    background-color: #ffffff;
}

Pygments

For the pelican-bootstrap3 theme. There are some settings that can be added to the pelicanconf.py file. See here. In particular:

PYGMENTS_STYLE = 'manni'

changes the pygments css. Most themes have the pygments css built in.

Flow Text around image

Use html in markdown file.

<img align="left" style="margin: 0px 20px" alt="parasled" src="/images/parasled.jpg?w=121" width="278" height="686" /></a>

Links to an anchor in another file: eg bdpq

requires at place to be linked TO: in file symmetry_tile_docs.md

<a name="bdpq"></a>

at place to be linked FROM:

["bdpq" notation]({filename}../symmetry_tile_docs.md#bdpq)

This only works in Pelican 3.4
